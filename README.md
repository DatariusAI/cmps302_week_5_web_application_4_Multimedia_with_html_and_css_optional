An Alpine tourism website incorporating multimedia elements with HTML and CSS. Features a photo gallery of Alpine countries and styled tourism content.

## Topics Practiced

- Embedding multimedia content in HTML pages
- Image gallery layout with CSS styling
- Multi-page site structure (`index.html`, `tourism.html`)
- Organized asset management (`Pictures/`, `Pictures/Gallery/`)
- CSS styling via external stylesheet

## Project Structure

```
index.html              # Home page
tourism.html            # Tourism page
css/styles.css          # Stylesheet
Pictures/               # Content images
Pictures/Gallery/       # Gallery images (8 Alpine countries)
```

## How to Run

Open `index.html` in any web browser.

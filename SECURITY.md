# Security Policy

## Reporting a Vulnerability

Please report suspected security issues via [GitHub Private Vulnerability Reporting](https://github.com/DatariusAI/cmps302_week_5_web_application_4_Multimedia_with_html_and_css_optional/security/advisories/new).

Do not open a public issue for security bugs.

## Supported Versions

Only the latest `main` branch is supported.
